Odoo All Sale Order, Purchase Order Cancel/Reverse

This module is used cancel sale/purchase/picking/delivary/Inventory Adjustments order in confirm state and also related invoices or incoming shipment/delivery orders of that order in odoo. We can easily cancel and reset the invertory adjustments as well.This module also have feature of cancel the picking and set it to draft stage specifically in odoo.

Features

Sale Order/Purchase Order Cancel

Option of canceling the sale order/purchase order in confirm stage.

Picking Cancel on Done stage

Option of canceling the picking in done stage.

Delete picking order.

Cancel the picking the picking will be in draft stage. so you can delete that picking order.

Cancel/Remove Generated Quants

Cancelled picking will also effect on generated quants.

Cancel/Remove Generated Moves

Cancelled picking will also effect on generated stock moves.

Inventory Adjustment By Lot/Serial Number

Cancel/reset product of lot/serial number.
